# Competition Merger Analysis – South Africa

## Project Overview

This project investigates the potential competition effects of a hypothetical merger between Shoprite and Pick n Pay in South Africa.

The analysis is conducted using principles from competition economics and industrial organization, with R used for data cleaning, analysis, and visualization.

## Research Question

Would a hypothetical merger between Shoprite and Pick n Pay substantially lessen competition in the South African grocery retail market?

## Objectives

- Define the relevant market
- Calculate market shares
- Compute concentration ratios (CR4 and CR5)
- Calculate the Herfindahl-Hirschman Index (HHI)
- Assess the potential effects of the merger
- Evaluate implications for consumer welfare
